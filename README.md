# Explainable-Multi-Modal-Hybrid-Neural-Network-for-Boundary-Aware-Brain-Tumor-Segmentation
A Hybrid 3D CNN + Transformer model for brain tumor segmentation on BraTS 2021 (1,251 patients). Features attention fusion, boundary-aware loss, calibration, ablation study, and SHAP explainability. Trained on multi-modal MRI (T1, T1ce, T2, FLAIR) with Dice, IoU, and Hausdorff evaluation.
